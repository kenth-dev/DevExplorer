# DevExplorer

DevExplorer is an interactive, educational web application designed for Senior High School students to explore programming languages and discover related career paths. It simulates a VS Code-like IDE, allowing users to view and compare "Hello World" programs in popular languages, and learn where each language is used in the real world.

---

## Overview
DevExplorer demystifies programming for students by showing real code, syntax highlighting, and career connections in a visually engaging, cyberpunk-themed interface. The app is fully responsive and works on both desktop and mobile devices.

- **Target Audience:** Senior High School students (ages 15–18), no prior programming experience required.
- **Goal:** Make programming approachable, help students identify languages and career paths, and inspire the next generation of developers.

---

## Features
- Explore "Hello World" code in different programming languages
- Switch between files using tabs (like an IDE)
- Learn what each language is used for
- Explore real-world career paths and which languages they use
- Works on desktop and mobile
- Syntax highlighting, word wrap, and keyboard navigation
